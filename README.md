# stock-market-prediction

This repository contains a Streamlit app for stock/index/commodity forecasting.

## Files
- `streamlitapp.py` — Streamlit application entrypoint
- `requirements.txt` — Python dependencies

---

## Deploy to Streamlit Community Cloud (Share)
1. Go to https://share.streamlit.io and sign in with GitHub.
2. Click **New app** → Choose repository **Ananya22-hub/stock-market-prediction**.
3. Select branch: `main` and file path: `streamlitapp.py`.
4. Click **Deploy**.

After the app is deployed you'll get a public URL like:

`https://share.streamlit.io/Ananya22-hub/stock-market-prediction/main/streamlitapp.py`

You can add the link and the badge below to this `README.md` after successful deploy:

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/Ananya22-hub/stock-market-prediction/main/streamlitapp.py)

- If you need any secrets (API keys or tokens), add them in the app settings on Streamlit Cloud (App → Settings → Secrets) rather than storing them in the repo.
